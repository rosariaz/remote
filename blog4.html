Blog Post 4


Combining falling text, animation, translate, and rotate we can produce an overlapping effect over a page. This video provides a tutorial on how to do so using a falling leaf to give an autumn effect. There aren't many opportunities to place this effect on a page as it can be distracting towards the main source material of the site. On the other hand, placing an effect on a main page can be inviting and pretty aesthetic to the user. I recreated a similar effect with red rose petals on this blog page.


This video showed how to make text appear when hovering images. This is very effective effect that can be used as a caption replacement. Less words on the screen create a more minimalistic look that appeals to visual learners. This technique can be applied to products on a site that flip and show the price, definition objects, flashcards, etc. I applied a similar effect on this text that flips into a picture.
