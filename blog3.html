This video provided a tutorial on a simple animation for text when loading up or refreshing a webpage. I recreated the same animation within my css file which can be seen on this text when refreshing the page. Although the animation is barely noticeable, it provides a smoother transition subconsciously as opposed to the bland and sudden load up. Soft and smoother animations allow the user to feel more relaxed and pleased when viewing the page.


Using animations, we can combine other techniques such as hover to add a multiple layered effect. While the video showed demonstrates a complicated animation into a glowing object when hovered, I made a button that pulsates when hovered. This effect can be used to bring attention and make the page more interactive for the user. Potentially, a business site could advertise their most desired product by making it animate constantly thus bringing the customer's attention towards it.
